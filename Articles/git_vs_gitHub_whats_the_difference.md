## Many times GitHub is confused with Git.  
and I understand that well, because the names look a little similar, lol.

But actually, the two are very different. Let’s understand this with an analogy.

Confusing Git and GitHub is like confusing a video file on your PC with YouTube.

Just like YouTube hosts `.mp4` videos, it lets you host, share, and distribute videos. It also works like a social network, where people can comment on videos, like them, and more.

Following the same idea, GitHub is like YouTube, but for Git files. On GitHub, you can host Git repositories, share them, distribute them, and it also has social features like pull requests and much more.

And in the same way that on YouTube you have an account to host your videos, on GitHub we also create an account to host our Git repositories. **The beginning of a big journey!**

When you create your GitHub account with your *username*, people can access it through the address `github.com/username`, or mention you with `@username`.

I recommend choosing a good name for your GitHub account, because when you create your username, even though it is possible to change it later, many references that point to your repositories, your profile, and other very important things can break. 🤝
