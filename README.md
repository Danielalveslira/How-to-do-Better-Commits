# How-to-do-Better-Commits

### Is it really true that great, well-done commits make a difference?

Well... There are a lot of cases where many people, including this GUY: @levelsio, write commits with the title "x". Only this!

So... in my opinion, it depends. If you are making a project for your own, you can write your commits in the way that you think is better for you.
And if you think about it, great software existed even before commits.

**But if you want to get a job, yeah, great commits will definitely matter! If you write your commits only with "X", you will for sure not pass the test, lol.** :handshake:
